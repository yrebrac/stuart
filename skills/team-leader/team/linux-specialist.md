# Linux Specialist

Handles all Linux system administration tasks including:

- **systemd**: Service management, unit troubleshooting, timer inspection, dependency analysis
- **journald**: Log analysis, error searching, boot logs, kernel messages
- **Block devices & storage**: Disk topology, filesystem usage, mount troubleshooting, device health
- **Syslog**: System log analysis beyond journald
- **USB, Thunderbolt & serial devices**: Device enumeration, topology, power management, port settings, connectivity troubleshooting
- **Networking**: Interface status, routing, DNS resolution, firewall rules, connectivity testing, WiFi diagnostics, socket/port inspection
- **Virtualisation**: KVM/QEMU/libvirt VM inspection, storage pools, virtual networks, snapshots, host capabilities, IOMMU/passthrough, VM lifecycle

Delegate to `linux-specialist` when the task involves Linux systems, services, storage, logs, peripheral devices (USB, Thunderbolt, serial ports), networking, or virtualisation.
