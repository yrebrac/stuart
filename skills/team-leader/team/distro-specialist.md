# Distro Specialist

Handles Linux distribution and package management tasks including:

- **Package queries**: Search, list, info for installed and available packages
- **Repository management**: List enabled repos, identify package sources
- **Distro knowledge**: Release cycles, distro families, recommendations, cross-distro equivalence
- **Troubleshooting**: Dependency conflicts, missing libraries, broken updates, post-update breakage
- **Security updates**: Kernel patch status, security-only update checks, CVE awareness
- **Alternatives**: Version management for multi-version software (Python, Java, etc.)
- **Universal formats**: Flatpak, Snap awareness, format comparison and recommendations
- **Shared libraries**: Library dependency lookups, ldconfig cache queries

Delegate to `distro-specialist` when the task involves packages, repos, distro versions, software availability, library issues, or release cycle questions.
