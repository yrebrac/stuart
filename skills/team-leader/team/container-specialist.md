# Container Specialist

Handles all container administration tasks including:

- **Docker/Podman**: Container lifecycle, inspection, logs, resource usage
- **Images**: Available images, disk usage
- **Networking**: Container networks, port mappings
- **Volumes**: Persistent data stores
- **Compose**: Stack status, service logs, multi-container applications

Delegate to `container-specialist` when the task involves containers, Docker, Podman, images, compose stacks, or container networking.
